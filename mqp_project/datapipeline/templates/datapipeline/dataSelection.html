Understanding the Number Next to theName

The number after the name is used to indicate which Study it corresponds to. When looking at the displayed studies that were selected, you can see the study's id. So if we have a name like "HeartRate_1" or "Control (1)", the instance of the "1" is used to indicate that these Data Categories or Study Groups correspond to the Study with the ID of 1.
